## ⚡ Guía de inicio rápido

Sigue estos pasos para obtener una copia local del repositorio en tu computadora:

**1. Abre una terminal** (CMD, PowerShell, o la terminal integrada de VS Code).

**2. Navega hasta la carpeta donde deseas guardar el proyecto:**

```bash
cd ruta/de/tu/carpeta
```

**3. Clona el repositorio con Git:**

```bash
git clone https://github.com/JorgeJuarez07/Curso-basico-de-Python.git
```

**4. Entra al directorio del proyecto:**

```bash
cd Curso-basico-de-Python
```

**5. ¡Listo!** Ya puedes abrir la carpeta en VS Code:

```bash
code .
```
